# Support

Need help with Colveor? Here's where to go.

## Documentation

Documentation and installation guides will be available as the project evolves. Start with the [`colveor-docs`](https://github.com/colveor/colveor-docs) repository and the [`colveor-examples`](https://github.com/colveor/colveor-examples) sample applications.

## Questions & Discussions

For questions, ideas, and general conversation, please use **GitHub Discussions** on the relevant repository.

## Bugs & Feature Requests

- 🐛 Found a bug? [Open a bug report](../../issues/new?template=bug_report.md).
- 💡 Have an idea? [Open a feature request](../../issues/new?template=feature_request.md).

Please search existing issues first to avoid duplicates.

## Finding the Right Repository

| Repository | Purpose |
|------------|---------|
| `colveor` | Project hub and roadmap |
| `colveor-backend` | Backend platform |
| `colveor-flutter` | Flutter framework and packages |
| `colveor-admin` | Administration panel |
| `colveor-cli` | Command-line tools |
| `colveor-docs` | Documentation |
| `colveor-starter` | Starter templates |
| `colveor-examples` | Sample applications |

## Security Issues

For security vulnerabilities, please follow our [Security Policy](SECURITY.md) rather than opening a public issue.
