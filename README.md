# Colveor `.github`

This repository holds the community health files and shared configuration for the [Colveor](https://github.com/colveor) organization.

**Colveor** is an open-source application platform for building scalable, production-ready software using a unified ecosystem of backend services, Flutter, web administration, and developer tooling.

## What's in here?

- 🙋 [`profile/README.md`](profile/README.md) — Organization profile shown on the Colveor GitHub page
- 🤝 [`CONTRIBUTING.md`](CONTRIBUTING.md) — How to contribute across Colveor repositories
- 📜 [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) — Community standards and expected behavior
- 🔐 [`SECURITY.md`](SECURITY.md) — How to report security vulnerabilities
- 💬 [`SUPPORT.md`](SUPPORT.md) — Where to get help
- 🧩 [`ISSUE_TEMPLATE/`](ISSUE_TEMPLATE) — Default issue templates
- ✅ [`PULL_REQUEST_TEMPLATE.md`](PULL_REQUEST_TEMPLATE.md) — Default pull request template

These files act as defaults for every repository in the Colveor organization that doesn't define its own.
